# 🧠 Quiz App Pro

A modern and interactive Quiz Application built using HTML, CSS, and JavaScript.

## 🚀 Features

- ⏱ Timer-based questions
- 📊 Progress tracking
- ✅ Option selection highlight
- 🔁 Restart quiz functionality
- 📱 Responsive design
- 🎨 Modern UI (Glassmorphism)

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
